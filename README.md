# Task-MVC

**Task-MVC** is a simple project developed for the DEPI initiative. It showcases static student cards with their images and details retrieved from a database. This project utilizes the ASP.NET MVC framework and demonstrates basic CRUD operations and database integration.

---

## Features

- **Student Card Display**:
  - View student details such as name, image, and associated information.
- **Database Integration**:
  - Fetch and display student data from a database.
- **Static Content**:
  - Predefined student data rendered in a clean and responsive layout.
- **MVC Architecture**:
  - Implements the Model-View-Controller pattern for separation of concerns.

---

## Technologies Used

- **Backend**: ASP.NET MVC (C#)
- **Frontend**: HTML, CSS, JavaScript
- **Database**: SQL Server
- **Styling**: Responsive CSS for a modern interface

---

## Installation

### Prerequisites

Ensure you have the following installed on your system:

- Visual Studio (with ASP.NET support)
- SQL Server
- .NET Framework

### Steps to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/ahmed3bdelaziz/Task-MVC.git
   ```
2. Open the solution file (`Task-MVC.sln`) in Visual Studio.
3. Configure the database connection string in the `Web.config` or `appsettings.json` file.
4. Run the database migrations (if applicable) to set up the required schema.
5. Build and run the project in Visual Studio.

---

## Usage

1. Launch the application locally or on a server.
2. Navigate to the section displaying student cards.
3. View static student details, including images and information stored in the database.

---

## Project Purpose

This project was developed as a simple demonstration for the DEPI initiative to showcase:
- Basic ASP.NET MVC functionality.
- Integration of a database-backed application.
- Display of static content in a user-friendly interface.
